# Smart Bin AI

## Overview

Smart Bin AI is an intelligent, real-time waste detection system designed to improve recycling and waste disposal practices. The system leverages computer vision and artificial intelligence to identify multiple waste items, assess contamination levels, and provide clear disposal guidelines.

The application is developed to promote sustainable habits through an accessible and user-friendly interface.


## Features

- Real-time multi-object waste detection  
- Contamination assessment (Dry, Mildly Dirty, Wet)  
- Automated disposal guidelines for detected items  
- Filtering of non-relevant objects  
- Lightweight and efficient performance  


## Technologies Used

- Python  
- Streamlit  
- YOLOv8 (Ultralytics)  
- OpenCV  
- NumPy  
- Pillow  


## Working

The system captures input through a camera, processes the image using an object detection model, filters relevant waste objects, evaluates contamination levels, and provides appropriate disposal instructions in real time.


## Applications

- Smart waste bins for homes, schools, and offices  
- Environmental education tools  
- Waste management systems for public spaces  
- Integration with IoT-based smart city solutions
